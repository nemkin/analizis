# Koordinátarendszerek

## Jakobi determináns számolása

|   | r |            $\vartheta$            | $\varphi$ |
| - | - | --------------------------------- | --------- |
| x |   |                                   |           |
| y |   | $\frac{\partial{}i}{\partial{}j}$ |           |
| z |   |                                   |           |

## Gömbi koordináták

$x = r\cdot{}sin(\vartheta)\cdot{}cos(\varphi)$  
$y = r\cdot{}sin(\vartheta)\cdot{}sin(\varphi)$  
$z = r\cdot{}cos(\vartheta)$

$det(J) = r^2sin(\vartheta)$
